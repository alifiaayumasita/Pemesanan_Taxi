# Aplikasi Pembuatan Order Laundry
Aplikasi ini akan menampilkan halaman web pemesanan taxi dan dapat menghitung tarif dengan melihat jarak

## Features
- Menampilkan hasil inputan.
- Menghitung total tagihan.
